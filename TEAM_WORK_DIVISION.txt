================================================================================
CONFLICT RESOLUTION
================================================================================

WHEN TEAMMATES ARE STUCK
--------------------------------------------------------------------------------
1. Ask them to explain the problem
2. Look at error messages together
3. Check documentation together
4. Google the error together
5. Pair program to solve it
6. Document the solution

WHEN DEADLINES ARE MISSED
--------------------------------------------------------------------------------
1. Understand why (too hard? unclear? personal issues?)
2. Adjust task difficulty
3. Provide more guidance
4. Extend deadline if needed
5. Reassign if necessary

WHEN CODE QUALITY IS LOW
--------------------------------------------------------------------------------
1. Don't merge - provide constructive feedback
2. Explain what needs improvement
3. Show examples of good code
4. Pair program to refactor together
5. Create coding standards document

================================================================================
FINAL TIPS FOR TEAM LEAD
================================================================================

1. BE PATIENT
   Remember: You were a beginner once too

2. EXPLAIN, DON'T JUST FIX
   Help them learn, don't just solve problems

3. ENCOURAGE QUESTIONS
   Create safe environment for asking

4. PROVIDE RESOURCES
   Share tutorials, articles, courses

5. PAIR PROGRAM REGULARLY
   Best way to teach and learn

6. GIVE CONSTRUCTIVE FEEDBACK
   Focus on improvement, not criticism

7. TRUST YOUR TEAM
   Let them make mistakes and learn

8. COMMUNICATE CLEARLY
   Set clear expectations and deadlines

9. LEAD BY EXAMPLE
   Write good code, good commits, good docs

10. CELEBRATE TOGETHER
    Success is a team effort!
